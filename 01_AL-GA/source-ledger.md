**Source-access note (citation grounding).** Every "Official Source URL" below points to an official state source: the state legislature's code portal, the state administrative-code publisher, or — for Arkansas and Georgia, whose official codes (Ark. Code Ann.; O.C.G.A.) are published exclusively through a state-sanctioned LexisNexis portal with no free section-level deep link — that official publisher portal (navigate by the citation in the adjacent column). Where an official site returned an access error during the original research (HTTP 403 / Cloudflare / DNS), the verbatim statutory text was read via a secondary republication (Justia/FindLaw/Public.Law) used only as a pathfinder; a dedicated correction pass on 2026-06-23 independently re-opened and confirmed the section text on the official source for Alabama, Alaska, Arizona, Colorado, and Connecticut. Tier reflects the rank of the governing authority itself (e.g., "1" = codified statute), not the access route.
